# prolog_microcredito

Para executar, siga os seguintes passos:

Baixar Prolog Portable: https://drive.google.com/file/d/1KetZAmsPVHPx-6kBvDNN4bUgp2vvT0kq/view?usp=classroom_web&authuser=0

Dentro da pasta do Prolog portable, clone este repositório: https://github.com/Albibino/prolog_microcredito

Inicie o executável SWI-PrologPortable.exe e rode os seguintes comandos:
  ['lpp_microcred_prolog/src/main.pl'] .
  make .
  start .

Após iniciar fique a vontade para testar a aplicação, não há necessidade de utilizar ( . ) ao final dos comandos.

Para validar a execução rode o seguinte comando após a primeira execução do programa: 
  explain:show_trail .
